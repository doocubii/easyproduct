# 미리보기 만들기 (UI 키트 · 샘플 페이지 · PNG 변환)

각 컨셉 안을 **눈으로 비교**할 수 있게, 안마다 HTML 두 장과 그 PNG를 만든다.
색상 코드만으로는 비전문가가 판단하지 못한다. 실제로 그려진 걸 보여주는 게 핵심이다.

## 만드는 파일 (안 하나당)
- `안{n}-{이름}-키트.html` — 컴포넌트 UI 키트
- `안{n}-{이름}-샘플.html` — 샘플 페이지 한 장
- 위 둘을 변환한 `.png` 2장
그리고 전체에 대해 `디자인안-비교.html` (세 안 PNG를 나란히 거는 표지) 1장.

## 공통 규칙
- 각 HTML 맨 위에 그 안의 **디자인 토큰을 CSS 변수로** 선언하고, 아래 요소는 전부 그 변수만 쓴다. 그래야 컨셉 문서 값과 안 어긋난다.
```html
<style>
  :root{
    --primary:#2563EB; --primary-strong:#1E40AF; --accent:#F97316;
    --bg:#FFFFFF; --surface:#F8FAFC; --text:#1F2937; --text-sub:#6B7280;
    --success:#16A34A; --warn:#F59E0B; --danger:#DC2626;
    --radius:10px; --shadow:0 1px 3px rgba(0,0,0,.08);
    --font: "Pretendard","Apple SD Gothic Neo",system-ui,sans-serif;
  }
</style>
```
- 폰트는 설치가 없어도 깨지지 않게 **시스템 폰트 fallback**을 항상 넣는다(위 `--font`). 웹폰트를 CDN으로 부를 수 있으면 링크를 넣되, 없어도 되도록.
- **한 파일 안에** 스타일까지 담는다. 외부 빌드·서버 불필요. 배경 흰색(또는 그 안의 배경색) 고정 — PNG가 투명하게 나오지 않도록.
- 다크모드를 지원하는 안이면 키트 하단에 어두운 배경 섹션을 하나 더 둔다.

## UI 키트에 넣는 것 (범용 — 특정 화면 아님)
1. **팔레트** — 색마다 색칠된 네모 + 이름 + hex + 용도 한 줄.
2. **버튼** — 기본 / 보조(테두리) / 비활성 / 위험(삭제 등), 보통·hover·눌림 느낌.
3. **입력창** — 기본 · 포커스 · 오류(빨간 테두리 + 안내문).
4. **카드** — 제목 · 본문 · 버튼이 든 카드 1~2개.
5. **배지·알림** — 성공/경고/오류 배지, 알림 배너.
6. **글꼴 위계** — 큰제목·제목·소제목·본문·작은 글씨를 실제 크기로 나열.

## 샘플 페이지에 넣는 것 (느낌 샘플 — IA 조립 아님)
- 상단 헤더(로고 자리 + 메뉴 2~3개 + 주요 버튼)
- 히어로 또는 대표 콘텐츠 영역(제목 + 설명 + CTA 버튼)
- 카드 목록 3개 정도
- 하단 푸터 간단히
- **주의**: 실제 IA 메뉴·기능을 그대로 조립하지 않는다. IA가 있으면 라벨만 살짝 빌려 사실감을 준다. 진짜 화면 설계는 다음 단계(별도 스킬)의 몫이다.

## PNG 변환
HTML을 그대로 이미지로 만든다. `scripts/render_png.py`를 쓴다.
```bash
python scripts/render_png.py 안1-안정형-키트.html 안1-안정형-샘플.html   # 여러 개 한 번에
# 결과: 같은 이름의 .png 생성 (기본 폭 1200px)
```

> ⚠️ **wkhtmltoimage는 이 스킬의 렌더 경로로 쓰지 않는다.**
> 이 스킬의 HTML은 디자인 토큰을 전부 CSS 변수(`var(--primary)` 등)로 쓴다(위 "공통 규칙").
> wkhtmltoimage(구형 WebKit 기반)는 CSS 변수를 지원하지 않아, 이 HTML을 wkhtmltoimage로 렌더하면
> 색·값이 전혀 반영되지 않고 **PNG가 백지로 나온다.** 그러므로 PNG 변환은 반드시
> `scripts/render_png.py`(Playwright / headless Chromium)를 거친다. 다른 렌더러로 대체하고 싶어도,
> **CSS 변수를 지원하지 않는 렌더러는 이 스킬의 HTML과 근본적으로 안 맞으므로 쓰지 않는다.**

- 이 스크립트는 Playwright(headless Chromium)로 렌더한다. 없으면 자동 설치를 시도하고, 그래도 안 되면 **PNG를 건너뛰고 HTML만 남긴 뒤 그 사실을 알린다**(배치가 통째로 실패하지 않게).
- claude.ai 등 변환기를 못 쓰는 환경이면 PNG 없이 HTML 미리보기로 대체하고 한 줄로 안내한다.

## 파일 이름 규칙
`안1-안정형`, `안2-활기형`, `안3-미니멀형`처럼 **안 번호 + 짧은 성격 이름**을 접두어로. 1안은 항상 추천안.
